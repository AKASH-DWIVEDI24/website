Etherpads
We use Etherpads for a shared, community editable document system for notes, call agendas, and more. Create an account to log in or use it as a guest. Create a new pad with simply typing “etherpad.hillhacks.in/p/(nameofthepad)” into your web browser. This server is currently managed by Arthur (kristof@sigsegv.be)
